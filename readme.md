# BusCleanliness

## Prerequisites
- python3
- pip3
- tkinter 
- json > for Data Store and retrive

## Installation
#### linux (arch)
```bash
$ sudo pacman -S python-tk
```
#### linux (ubuntu)
```bash
$ sudo apt-get install python3-tk
```
### Download and extract the pt3.tar.gz file
```
$ tar -xzvf pt3.tar.gz -C /yourDirectory
```
## Usage
```bash
$ python3 clean.py
```
